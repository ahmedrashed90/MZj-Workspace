إدارة التاسكات والحملات

أنشئ أجندات أو حملات + كروت (Columns). افتح أي بورد واضف تاسكات مرتبطة بـ Unique Spec Key (بدون الألوان الداخلية/الخارجية).

الأجنداتBoards

كل أجندة = بورد فيه كروت. افتح البورد لإضافة تاسكات.

مفيش أجندات لسه اضغط "إنشاء أجندة" وحدد الكروت (مثال: تصوير / مونتاج / مراجعة / نشر).

الحملاتCampaigns

الحملة = اسم + وصف + فترة + كروت. افتح الحملة لإضافة تاسكات وروابط.

مفيش حملات لسه اضغط "إنشاء حملة" وحدد تاريخ البداية والنهاية والكروت.

إنشاء أجندة

اكتب اسم الأجندة وأضف الكروت (Columns).

أضف كل كارت مرة واحدة. تقدر تحذف أي كارت قبل الحفظ.
سيتم حفظ الأجندة في Firestore: collection = agendas

إنشاء حملة

حدد بيانات الحملة وفترتها ثم أضف الكروت.

الكروت دي هتبقى الأعمدة الخاصة بالحملة.
سيتم حفظ الحملة في Firestore: collection = campaigns

إضافة تاسك

اختار Unique Spec Key (بدون الألوان) + وصف + موعد + المسؤول.

المصدر: collection = media_specs (سيتم عرض السيارة + البيان + الموديل فقط).
المصدر: collection = users (ولو فاضية بيظهر ديفولت).
مثال: رابط تصميم Canva / Drive / Brief / إعلان…
سيتم حفظ التاسك في Firestore: collection = tasks
